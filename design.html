08  克制的动效，严肃的无障碍

一套编排的错峰入场 + 少量微交互；尊重 prefers-reduced-motion。无障碍对标 WCAG 2.2 AA。

编排式错峰入场 / Staggered Reveal

滚动进入视口时，子元素以 ~60ms 递增延迟依次浮现（IntersectionObserver 驱动）。

微交互 / Micro-interactions

导航青色下划线扫出、卡片上浮、按钮位移——全部短促（150–300ms）、可预期、不喧宾夺主。

减弱动效 / Reduced Motion

@media (prefers-reduced-motion: reduce) → 动画时长归零、reveal 直接可见、扫描线静止。

可见焦点环

键盘焦点 2px 实色 --focus 环 + 偏移，绝不移除 outline。

:focus-visible

44px+ 触控目标

按钮、切换键最小高度 ≥ 44–46px，满足触控可达性。

语义化 HTML

header / main / section / article / nav 结构，aria-label 完整。

对比度纪律

正文用 #006e93 / #14191d 而非 petrol #009999——后者对比不足，仅作字标与图形。

技术 / TECH STACK

09  无构建 · 数据驱动 · 随处可部署

纯静态 HTML/CSS/JS，无打包步骤；内容由 data.js 驱动；全相对路径，扔到任何静态主机即可上线。

无构建静态站

原生 HTML/CSS/JS，零依赖、零编译，可读可改可审计。

数据驱动

课程 / 路径 / 反馈集中在 window.SITE_DATA，页面按需渲染。

相对路径

快速加载
