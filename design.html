轻量路由 / LIGHTWEIGHT ROUTING

04  没有后端，也能"路由"

不引服务器、不上框架路由。靠跨页链接、顶部锚点导航和 URL 查询参数，就完成了页面间的状态传递。

catalog.html?track=track-simatic-tia
预筛路径：目录页读取 ?track=，进入即把路径筛选器锁定到该路径，无需用户手动翻找。URL 参数
course.html?id=TIA-201
定位课程：课程页读取 ?id= 在数据源中查到对应课程并渲染全部内容与面包屑。URL 参数
index.html#paths · #resources · #voices
锚点导航：顶部导航与页脚用同页锚点平滑滚动，跨页则回到首页对应分区。锚点

链接即接口 · 状态写在 URL 里，可收藏、可分享、可直达 —— 零路由库

双重架构 / TWO ARCHITECTURES

05  前台发现漏斗 · 后台 App-Shell

两种截然不同的导航范式，却说同一种设计语言——共用 styles.css 的一套设计令牌与明暗主题。

公开站 · 顶部导航

index · catalog · course
  • 顶部水平导航 + 锚点，引导式 发现漏斗
  • 长滚动落地页，分区叙事：路径 → 支柱 → 课程 → 反馈
  • 移动端抽屉菜单，响应式栅格

管理后台 · 侧栏 App-Shell

admin · 客户端视图切换
  • 固定侧栏 + 顶栏， 视图切换（data-view，不整页刷新）
  • 总览 KPI/图表、课程 CRUD、学员/路径/资料/设置
  • 课程改动持久化到 localStorage（si-admin-courses-v1）
共用一套设计令牌 · var(--primary) / --font-mono / .card / .btn / 明暗主题
纯静态 / NO BUILD

06  为什么是纯静态、无构建

原生 HTML/CSS/JS，没有打包、没有编译。换来的是这几件实在的好处。

秒开

仅两个 JS、一份 CSS，无运行时框架，首屏轻量、即点即开。

可移植

assets/ 全相对引用，可放任意子目录，不锁死域名。

任意静态托管

GitHub Pages / Gitee Pages / Netlify / 任意 CDN / 本地 file:// 都能跑。

交付即拉即用

零依赖、零编译，协作者拉下来直接 serve，可读、可改、可审计。

李
PERSONA · 学习者画像
李工自动化工程师
想系统掌握 TIA 博途 / S7-1500 编程 车间一线 · 看得懂电气图 用碎片时间学习 常在手机/平板上看

目标很明确：不要零散视频，要一条能从入门走到精通的路径。车间光线常常很暗，需要护眼；时间被设备调试切得很碎，每次只能学一小段，所以"快速找到该学的那一门 + 学起来不分心"对他最重要。

旅程时间线 / JOURNEY

08  李工的六步旅程

每一步拆成三栏：我做什么 · 页面给我什么 · 体验为什么好。

进入首页

index.html
我做什么

第一次打开站点，想先搞清"这到底是什么、靠不靠谱"。

页面给我什么

英雄区一句话定位 + 实时工业控制台面板（PLANT-01 读数），下方 6 条学习路径卡。

为什么好

3 秒看懂"这是西门子工业培训"，控制台传达专业可信，路径让我立刻对号入座。

选一条路径

→ catalog.html?track=
我做什么

点「SIMATIC S7 与 TIA 博途」这条路径上的"查看课程"。

页面给我什么

自动跳到课程目录，并已用 ?track= 把路径筛选器预选到这条路径。

为什么好

意图被一路带过去，落地即是相关结果，不用自己重新翻找。

找到那门课

catalog.html
我做什么

在路径内再按等级筛"入门"，或直接搜课程名/编号。

页面给我什么

路径×等级×搜索三重过滤 + 实时计数；课程卡上编号/等级/学时/形式一目了然。

为什么好

卡片信息密度刚好，几下就锁定 TIA-201，决策成本很低。

开始学习

→ course.html?id=TIA-201
我做什么

点开课程卡，进入学习页正式上课。

页面给我什么

视频播放器 + 章节目录 + 「你将学到」+ 讲师卡 + 侧栏报名/进度，一屏聚齐。

为什么好

所有学习要素聚拢在一处，视线不用到处跳，沉浸、不分心。

持续推进

course.html
我做什么

点不同章节、看自己学到哪、想着学完接哪门。

页面给我什么

点章节即联动播放器标题/进度；进度条可视；侧栏推荐同路径下一门课。

为什么好

有掌控感与即时反馈，学完一门顺势接下一门，路径不断档。

适配场景

全站 · si-theme
我做什么

车间灯光暗，想护眼；换到平板/手机接着看。

页面给我什么

右上角一键切「暗色驾驶舱」主题，偏好存进 si-theme 全站通用；响应式照常。

为什么好

暗光护眼、跨设备一致、刷新不丢设置，碎片时间也能舒服地学。
